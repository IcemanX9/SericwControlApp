Policy Information

Internal Fly Control

Should Electronic Fly Units (EFK) be used:

  • Record activity found in the EFK on the activity report in the pest control file.
  • Remove glue board. Clean out the unit and place used boards in a bucket to be disposed of.
  • Insert new board.
  • Clean tube using a clean cloth.
  • Replace tubes on a yearly basis. All damaged tubes should be replaced immediately.

All services will be recorded in the pest control file under the applicable sections.
